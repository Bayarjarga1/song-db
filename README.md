# Song Database Management System

Song Database Management System is a menu-driven program written in C.  
The program allows users to store and manage songs dynamically using linked lists and file handling.

## Features

The program provides the following functionalities:

- Add a new song
- Display all songs
- Search songs by title
- Delete songs
- Save database to file
- Load database from file
- Exit program

Each song contains:

- Title
- Artist
- Album
- Release Year
- Genre
- Duration (minutes and seconds)

## Data Structures and Design

The program is implemented using:

- Singly linked list
- Dynamic memory allocation
- Structures
- File handling
- Menu-driven interface using switch statements

The linked list is used to dynamically store song records during runtime.

## File Handling

The program stores song information in:

```text
song_database.txt
```
